# valksor-logo

Valksor logo

## Examples

### Default

```html
<valksor-logo></valksor-logo>
```

### Set size

```html
<valksor-logo size="96px"></valksor-logo>
```

### Set width and height separately

```html
<valksor-logo width="120px" height="80px"></valksor-logo>
```
